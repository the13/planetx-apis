SKplanet Developer Center API code sample

Tmap, Tcloud, Weather planet의 API code sample을 포함한다. tab을 눌러 시험 사용해 볼 수 있다.

T map API

현재위치(blue)에서 주변 편의점(red)을 검색하고, red marker를 click시 보행자 경로를 안내, 지도 click시 주소 안내

  • 주변 편의점 검색: POI 주변검색 API
  • 보행자 경로안내: 보행자 길찾기 API
  • 주소 표시: reverse geocoding API

T cloud API

T cloud에 file upload, 사진 thumbnail 목록

  • Tcloud login/logout: OAuth 인증
  • Tcloud upload: 업로드 토큰 조회 API
  • Tcloud thumbnails: 사진 목록 조회 및 검색 API

Weather Planet API

현재위치 날씨정보 제공

  • 현재 지역날씨: 날씨정보 현재날씨 API
